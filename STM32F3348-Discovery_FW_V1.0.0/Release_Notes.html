Update History

  

The STM32F3348 Discovery Board Firmware Applications Package provides ready-to-run firmware examples to support quick evaluation and development on STM32F3348 DISCO board. It support the following examples:
  • STM32F3348 Discovery peripheral firmware examples, under Project\Peripheral_Examples, for more details about each application refer to associated readme file.
  • STM32F3348 Discovery Board Demonstration firmware, under Projects\Demonstration, for more details about this application refer to associated readme file.
For more information on the STM32F3348 Discovery, to download the available firmware package and documentation visit www.st.com/stm32f3-discovery.
