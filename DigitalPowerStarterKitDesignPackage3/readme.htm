The Digital Power Starter Kit Package contains the following:

 

1. Information Sheet including schematic

2. DPSK 3D View

3. 2P2Z Peak Current Mode (PkCM) Source Code

4. Bill of Materials

5. License agreement

 

Important Notes:

 

1.  If the sticker on Digital Power Starter Kit reads “02-10312-2-R2 BUR152521030”, then make the following hardware changes on the Digital Power Starter Kit:

(Refer to dsPIC33EP_MPLAB_Starter_Kit_Info_Sheet - DS50002344B)   

a. C11 (shown in Figure.1) component is to be replaced with 10uF.

b. C26 (shown in Figure.2) component is to be replaced with 100pF.

c. C20 (shown in Figure.2) component is to be replaced with 240pF.

 

2.  If the sticker on Digital Power Starter Kit reads “02-10312-2-R3 BUR152521030”, then there is no need for any hardware modifications

 

3.  Use MPLAB XC16 compiler version v1.23 or above.
